# Band Name Generator

Welcome to the Band Name Generator! This is a simple Python program that suggests a band name based on your city and pet's name.

## Getting Started

To generate a band name, follow these steps:

1. Run the program.
2. When prompted, enter the name of the city you grew up in.
3. When prompted, enter your pet's name.
4. The program will suggest a band name based on your inputs.

## Prerequisites

- Python 3.x

## Usage

1. Open a terminal or command prompt.
2. Navigate to the directory where the code file is located.
3. Run the following command to execute the program:

   ```bash
   python band_name_generator.py
